    1.3 环境搭建¶

    上一章节中，我们已经了解了模型微调重要下，接下来我们在自己的电脑或云服务器上成功配置AI开发环境。

    1. 基础环境¶

    2.1 确认GPU支持¶

    确认是否拥有NVIDIA显卡。AMD显卡无法使用CUDA进行加速。

    在Windows系统中，右键点击桌面，如果能看到 “NVIDIA控制面板” 选项，则说明电脑配备了NVIDIA显卡。如下图所示：

    另外也可以通过任务管理器查询：

    或者通过设备管理器也可查看

    注意 ：没有NVIDA的显卡是不能使用CUDA的，也就不能使用GPU加速的，后续所有工具包需安装CPU版本。

    2.2 安装Python¶

    推荐使用 Python 3.10 或更高版本

    安装 Anaconda 或 Miniconda。它们能帮您轻松创建和管理独立的Python环境，避免包冲突。也可以从清华大学开源软件镜像站下载Anaconda以获得更快的速度

    2.安装NVIDIA驱动与CUDA工具包¶

    接下来我们需要安装以下应用：

    • NVIDIA驱动：硬件显卡驱动，是为了告诉系统如何使用这张显卡的。
    • CUDA：NVIDIA为了构建自己的护城河而开发的高性能计算库，主要用于3D渲染和深度学习等。
    • CUDNN：NVIDIA针对深度学习而开发的，在CUDA之上的深度学习工具库。

    就是如果你想加速你的AI程序的训练和推理速度，那么 CUDA 与 CUDNN 是必须的。

    开始之前，先检查自己的GPU是否为 Nvidia 显卡，如果是再接着进行以下的工作；如果不是，很遗憾，你的机器与 AI 基本绝缘了。查看方式如下：

    如果有NVIDIA显卡，但是没有安装驱动，那么就需要去 Nvidia 驱动安装来安装驱动，安装的过程跟普通程序安装差不多。进入该网站后选择适合自己显卡的驱动，然后点击搜索即可，如下图所示：

    下载好后就按照安装提示一步步安装即可。

    • 首先，打开 NVIDIA 控制面板，点击左下角的 “系统信息”：

    • 点击组件，就能看到 NVCUDA64.DLL 的驱动型号了，如下图所示，我的显卡所支持的 CUDA 最高型号为： 12.7.33，那么靠前的型号都支持。

    3.安装pytorch¶

    PyTorch 是一个基于软件的开源深度学习框架，用于构建神经网络，完成模型训练和预测。作为一款主流的深度学习框架，PyTorch 具备灵活性和易用性，支持多种 AI 模型训练与部署方式。这些优点使 PyTorch 成为学术和研究界领先的深度学习框架，也越来越多的被企业应用在自然语言处理、图像识别等 AI 场景中。

    3.1 创建虚拟环境¶

    为了避免包管理混乱，强烈建议为每个项目创建独立的虚拟环境

    conda create -n XXX python=3.10
conda activate XXX

    3.2 安装pytorch¶

    首先访问 PyTorch官网。根据操作系统、包管理器以及上一步安装的 CUDA版本，选择对应的安装命令。

    如果您使用CUDA 12.6和pip，官网给出如下命令：

    pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu126

    直接在终端安装即可。

    注意：尽量使用官网生成的命令，并确保PyTorch的CUDA版本与本地安装的CUDA t版本匹配。如果版本不匹配，可能会导致 CUDA not found 错误

    你也可以选择其他版本的torch：

    4.环境验证¶

    安装完pytorch后我们可以通过以下代码验证是否安装成功，及cuda的可用性：

    # 1. 导入必要的库
import torch

# 2. 检查PyTorch和CUDA
print(f"PyTorch版本: {torch.__version__}")
print(f"CUDA是否可用: {torch.cuda.is_available()}")

# 如果CUDA可用，进一步检查信息
if torch.cuda.is_available():
    print(f"当前GPU设备: {torch.cuda.get_device_name(0)}")
    print(f"CUDA版本: {torch.version.cuda}")

    输出以下内容说明安装成功：

    5. 总结¶

    通过上述内容完成了AI开发环境的搭建，包括安装NVIDIA显卡驱动、CUDA工具包和PyTorch深度学习框架，并提供了环境验证方法。我们成功搭建支持GPU加速的AI开发环境，为后续模型训练和微调做好准备。

    常见问题与解决方案

    • CUDA not found 错误：这是由于 PyTorch与本地CUDA版本不匹配 造成的。请在PyTorch官网，仔细核对并重新生成安装命令。
    • nvidia-smi 命令无法识别：说明NVIDIA显卡驱动未正确安装。请前往NVIDIA官网下载并安装适合您显卡的驱动。
    • 安装速度慢：可以使用国内镜像源（如清华源、阿里云源）来加速下载
